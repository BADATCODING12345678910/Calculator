# Simple Calculator Website

A clean, modern calculator web application with a responsive design.

## Features
- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Clean, modern UI
- Responsive design
- Error handling
- Keyboard support

## How to Use
1. Open `index.html` in any modern web browser
2. Use mouse clicks or keyboard to input numbers and operations
3. Press 'C' or click the clear button to reset
4. Press '=' or click the equals button to calculate

## Keyboard Support
- Numbers: 0-9
- Operations: +, -, *, /
- Equals: Enter or =
- Clear: C or Escape

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)

## Browser Support
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
